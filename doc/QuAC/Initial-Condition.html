7.3.8 Initial Condition

The initial condition of the free particle (the value of the function in the first point is not zero, See Overview,) can be used, by the keyword FPIniCond.

General syntax: 

FPIniCond

Incompatibilities:

There are no incompatibilities for this keyword. CHANGE THE BOUND—–> INWARD INTEGRATION WILL FAIL!!

Default values:

The initial condition of the free particle is used by default just for calculations on ‘FREE_PARTICLE’ or ‘Square’ potentials. In the other cases the derivative is the one of the free particle and the value of the function is set to zero at the initial integration point.
