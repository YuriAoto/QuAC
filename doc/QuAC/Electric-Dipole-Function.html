7.4.4 Electric Dipole Function

The electric dipole moment function for an intensity analysis can be given with the keyword ElecDipoleFunction.

General syntax: 

ElecDipoleFunction dip_name

dip_name is the name of the dipole moment function to be used in this analysis.

Incompatibilities:

Have meaning only for a VibrationalIntensity analysis. It is incompatible with the keyword FrankCondon. Default values:

There are no default value for this keyword. !!!CHANGE TO RECOGNISE IF ONLY ONE WAS GIVEN!!
