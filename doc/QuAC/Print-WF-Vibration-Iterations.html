7.3.22 Print WF Vibration Iterations

If the keyword PrintVibWFIterations is given, the wave function will be printed in every iteration for a BOUND_STATES calculation. The keyword Print (See Print,) can be used to define the interval that the wave function will be printed.

General syntax: 

PrintVibWFIterations

Incompatibilities:

Have meaning only for a BOUND_STATES calculation.

Default values:

The vibrational wave function for each iterations are not printed by default.
