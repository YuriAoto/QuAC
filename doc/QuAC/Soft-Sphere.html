4.2 Soft Sphere

SoftSphere define a soft sphere, which is also zero valued after a given distance, the radius, but below it has a repulsive exponential behaviour.

Expression:

V(r) = D((r_0/r)^a - 1) if r <= r_0
V(r) = 0.0 if r > r_0

General syntax: 

radius epsilon alpha

radius = r_0, epsilon = D, and alpha = a.
