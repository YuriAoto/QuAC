7.3.19 Vibrational Levels

The required vibrational levels for a BOUND_STATES calculation can be specified in two ways, similar to the specification of the angular momentum. (See Angular Momentum.)

General syntax: 

VibLevel
v_1
v_2
...
v_n

or 

VibLevelMax v_max

Here, v_1, v_2, v_n, and v_max are integers and correspond to the vibrational levels required. The second case is equivalent to ask for the levels 0, 1, ..., v_max by the first.

Incompatibilities:

Have meaning only for a BOUND_STATES calculation.

Default values:

If not passed in a vibrational calculation, QuAC attempt to calculate every vibrational level.
