7.13 Final Integration Step

The integration step used in the last part of the integration can be different from the one used in the interaction range, in order to speed the calculation. It is reached doubling the step size, starting from Rmax. (See R min and R max.) Is given using the keyword FinalIntStep and follow the same syntax of the IntStep keyword. !!!! BLEND WITH INTSTEP.

General syntax: 

FinalIntStep step

step is the step size in the end of the integration, in the unit used in the potential of the current job.

Incompatibilities:

It can be given only for PHASE_SHIFTS and SCATT_LENGTH calculations. Must be greater or equal the integration step.

Default values:

It default value is rho = kr = 0.005 , the same of the integration step, i.e., the step is not increase by default.
