7.3.15 Save Global Potential

In order to save computing time, the potential at every point needed in a calculation can be calculated just once and stored. This is useful specially for computing demanding potentials, like ‘Direct’ potential.

General syntax: 

SaveGlobalPot

Incompatibilities:

Not implemented for SCATT_LENGTH calculation, since only one calculation is carried. For a BOUND_STATES calculation require a fixed mesh points integration. (See Vibrational Fixed Mesh.)

Default values:

Do not save the potential by default.
