7.3.3 Integration Method

The numerical integration can be performed by fourth-order Runge-Kutta, Adams-Bashforth-Moulton predictor-corrector, Numerov, or Johnson log-derivative method. It is specified by the keyword IntMeth

General syntax: 

IntMeth int_method

Here int_method is the integration method and can be RK4, ABM, NUMEROV, or JOHNSON.

Incompatibilities:

The Johnson log-derivative method is implemented only for SCATT_LENGTH calculations by the Meshkov procedure, for which is required. (See Scattering Length.)

Default values:

The default integration method is the Numerov, with the exception of the Meshkov procedure for scattering length calculation, which requires the Johnson log-derivative method.
